# AsyncFortress 3D Clash

GitHub Pagesで公開できる、非同期対戦向けの3D要塞衝突バトルプロトタイプです。

## Prototype

- 5x4の要塞グリッドに砲台、ミサイル、盾、突撃槍、エンジン、装甲を配置します。
- 中央のコアは固定で、コアHPが0になると敗北します。
- 2つの要塞が自動で前進し、砲撃、ミサイル、衝突押し合いで戦います。
- 勝敗はコア破壊、押し戻し、時間切れ時のスコアで決まります。
- 要塞コードのエクスポートとライバルコードのインポートに対応しています。

## GitHub Pages

公開元は `gh-pages` ブランチの `/` です。
