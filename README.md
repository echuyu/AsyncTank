# AsyncRacer Swipe Mini

GitHub Pagesで公開できる、非同期対戦向けのミニ四駆風スワイプ改造レースプロトタイプです。

## Prototype

- 車体の6スロットにブースター、ローラー、グリップ、重り、ウイング、バンパーを装着します。
- パーツはタップまたはドラッグでスロットへ付け替えられます。
- 同じコースを自分のマシンとライバルのゴーストが走り、タイムで勝敗を決めます。
- ブースト区間、ジャンプ、荒れた路面、壁寄り区間があり、速度と安定性のバランスが結果に影響します。
- マシンコードのエクスポートとライバルコードのインポートに対応しています。

## GitHub Pages

公開元は `gh-pages` ブランチの `/` です。
