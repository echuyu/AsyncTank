# AsyncTank Katamari BOT

GitHub Pagesで公開できる、ベイブレード風3D塊BOTバトルの静的プロトタイプです。

## Prototype

- 球体コアの周囲18ソケットにパーツを貼り付けます。
- 目、噴射、トゲ、砲、盾、重りの配置でBOTの動きが変わります。
- 組み立て画面では推進方向、攻撃面、防御面、重心バランスを視覚メーターで確認できます。
- 傾斜つきスタジアムで1対1の自動戦闘をします。
- `Rapier 3D` によるWASM剛体物理で、重さ・反発・押し合い・パーツ形状が勝敗に影響します。
- 衝突、砲撃命中、リングアウト時に火花、波紋、ヒットストップ、カメラ揺れが入ります。
- 序盤は外周バンパーで跳ね返り、終盤は壁が下がってリングアウトが起きやすくなります。
- 外周から弾き出されるとリングアウトです。
- BOTコードのエクスポートとインポートに対応しています。
